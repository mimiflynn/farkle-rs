Farkle Scorecard
=====================

So you never need paper!

## Features

- List of rules
- Points reference
- Track Farkle rate!

## Tech

- React
- sessionStorage

## User Flow

- add names of players
- set players turn order
- players roll until on the board (500 or 1000 point options)

## Development

```
npm install
npm run start
```

Then point your browser to `http://localhost:3000`
